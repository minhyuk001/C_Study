## Math Grammer
```python
a = a + b
a = a - b
a = a * b
a = a / b
a = a % b

a += b
a -= b
a *= b
a /= b
a %= b
```

## Increase / Decrease
```
++num // 먼저 증가하고 대입
num++ // 대입하고 증가

--num // 먼저 감소하고 대입
num-- // 대입하고 감소
```

## Relational Operator
```
a == 1 // a와 1이 같은가?
a != 1 // a와 1이 다른가?
a > 1 // a가 1보다 큰가?
a < 1 // a가 1보다 작은가?
a >= 1 // a가 1보다 크거나 같은가?
a <= 1 // a가 1보다 작거나 같은가?

a && b // a와 b가 모두 참일 경우에만 참
a || b // a와 b가 모두 거짓일 경우에만 거짓
!a // a가 참이면 거짓 거짓이면 참
```
